How to Play

🎯 Objective

Destroy vulnerable code and malware while letting secure code pass through safely!

🎮 Controls

  • Desktop: Arrow keys or A/D to move, SPACE to shoot
  • Mobile: Drag to move, tap to shoot
  • View Code: Click/tap any code block to view the full code

📊 Code Types

  • Secure Code: Let it pass! Don't shoot it.
  • Vulnerable Code (⚠): Shoot it before it reaches the bottom!
  • Malware (☠): Very dangerous! Shoot it immediately!

💯 Scoring

  • Destroy Vulnerable Code: +10 points
  • Destroy Malware: +25 points
  • Shoot Secure Code: -15 points
  • Miss Vulnerable Code: -20 points
  • Miss Malware: -40 points
  • Let Secure Code Pass: +2 points

🚨 Game Over Conditions

  • Shoot 15 secure code blocks
  • Miss 10 vulnerabilities/malware combined

💡 Pro Tips

  • Click on code blocks to read them better - the game keeps running!
  • Build streaks by destroying multiple threats in a row for bonus points
  • The game gets faster and harder as you progress
  • Press 'E' to end early and review your mistakes
